Crystal Game!

    Instructions

  • Match the "Target" number below using the hidden values of the four crystals below.
  • Click a crystal to add each of the values to the "Score."
  • Player wins by matching the score with target amount.
    Player losses by score going over the target amount.
  • Remeber: Although the point value of each Crystal is hidden, the value can be determined by tracking the score difference after each player click.
  • Also each time the game starts, the values of each Crystal changes.
